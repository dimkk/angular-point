getMockData
service in module fieldService

Description

Can return mock data appropriate for the field type, by default it dynamically generates data but the staticValue param will instead return a hard coded type specific value

Dependencies

  • ChanceJS

    to produce dynamic data. https://github.com/victorquinn/chancejs

Usage

getMockData(fieldType, [options]);

Parameters

ParamTypeDetails
fieldTypestring

Field type from the field definition.

[options]object

Optional params.

options.staticValueboolean

Default to dynamically build mock data.

(default: false)

*

mockData
