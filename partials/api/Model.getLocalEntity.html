getLocalEntity
service in module Model

Description

Similar to Model.searchLocalCache but you don't need to specify a query, only searches by list item id, and returns a promise that is fulfilled once the requested list item is registered in the cache

Usage

getLocalEntity(entityId);

Parameters

ParamTypeDetails
entityIdnumber

The ListItem.id of the object.

Returns

promise

Will resolve once the item is registered in the cache.

Example

var task = {
   title: 'A Task',
   project: {
       lookupId: 4,
       lookupValue: 'Super Project'
   }
};

// Now we'd like to get the project referenced in the task
projectModel.getLocalEntity(task.project.lookupId).then(function(entity) {
    var projectThatICareAbout = entity;
    //Do something with it
}
