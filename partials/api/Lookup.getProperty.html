getProperty
service in module Lookup

Description

Returns a promise which resolves with the value of a property in the referenced object.

Usage

getProperty(propertyPath);

Parameters

ParamTypeDetails
propertyPathstring

The dot separated propertyPath.

Returns

promise

Resolves with the value, or undefined if it doesn't exists.

Example

var project = {
   title: 'Project 1',
   location: {
       lookupId: 5,
       lookupValue: 'Some Building'
   }
};

//To get the location.city
project.location.getProperty('city').then(function(city) {
   //Resolves with the city property from the referenced location entity

   });
